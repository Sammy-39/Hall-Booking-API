Hall Booking

API Endpoints

  • List all rooms with booked data - /hall-booking/rooms
  • List all customers with booked data - /hall-booking/customers
  • Create room (POST) with body {"no-of-seats": ${value},"amenities": [${value}], "price-per-hour" : ${value}} - /hall-booking/room
  • Create a booking in a room (POST) with body { "customer-name":${value}, "date":${value}, "start-time":${value}, "finish-time":${value}, "room-id":${value} } - /hall-booking/booking
